## 💡 프로젝트 개요
학사 일정, 강의평, 학교생활 꿀팁 등 재학생들에게 필요한 필수 정보를 대화형으로 제공하는 을지대학교 전용 AI 챗봇 시스템입니다. 파편화된 학내 정보를 한곳으로 모아 학생들의 정보 접근성을 높이고자 기획되었습니다.

## 🛠 사용 기술
*   **Backend:** Java, Spring Boot, OpenAI API
*   **Database:** MySQL

## 📌 핵심 기여도
*   을지대학교 학사정보시스템(EIS) API와 연동하여 사용자의 학적 정보를 기반으로 맞춤형 답변을 제공하는 백엔드 서버 로직 설계
*   OpenAI API를 활용한 프롬프트 엔지니어링 및 응답 결과 파싱/캐싱 처리
*   사용자 인증(Authentication) 및 학번/권한 매핑 로직 구현

## 🔥 트러블슈팅 및 성능 개선
**1. 외부 API(OpenAI, EIS) 호출로 인한 챗봇 응답 지연 해결**
*   **문제:** 챗봇에 질문을 입력하면, 내부 서버가 OpenAI API와 학교 학사정보시스템(EIS)을 순차적으로 호출하면서 사용자에게 답변이 도달하기까지 약 4~5초의 긴 대기 시간이 발생함.
*   **원인:** 동기(Synchronous) 방식으로 외부 API를 호출하여 네트워크 병목이 발생했고, 자주 묻는 질문(FAQ)에 대해서도 매번 새롭게 연산을 수행함.
*   **해결:** 
    1. 스프링의 `@Async`와 `CompletableFuture`를 활용하여 외부 API 호출을 비동기로 처리.
    2. '학사 일정', '도서관 운영 시간' 등 변동이 적고 반복적인 질문의 키워드를 분석하여 Spring Cache를 적용해 인메모리에 결과를 임시 저장함.
*   **결과:** 빈출 질문에 대한 응답 시간을 500ms 이내로 단축시키고, 외부 API 호출 비용(트래픽)을 약 40% 절감함.
