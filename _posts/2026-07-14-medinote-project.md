## 📌 핵심 담당 업무 (Key Roles)
1. **QR 코드 기반 데이터 매핑 API 개발:** 환자의 문진 데이터를 고유 ID와 매핑하여 빠르고 안전하게 조회할 수 있는 RESTful API 설계 및 구현
2. **데이터베이스 설계 (ERD):** 환자 정보, 문진 기록, 병원 정보 간의 관계형 데이터베이스 모델링 및 쿼리 최적화
3. **AWS 환경 구축 및 배포:** EC2를 활용한 서버 호스팅 및 RDS를 연동하여 안정적인 데이터 관리 환경 구축

## 🔥 트러블슈팅 (Troubleshooting)

### 1. [문제 상황 요약: 예) 대량의 문진 데이터 조회 시 응답 지연 현상]
*   **문제 인지:** 테스트 과정에서 환자의 과거 문진 기록을 불러올 때 조회 속도가 00ms 이상 지연되는 현상 발생.
*   **원인 분석:** 환자 테이블과 문진 기록 테이블을 JOIN 하는 과정에서 적절한 인덱스가 설정되어 있지 않아 풀 테이블 스캔(Full Table Scan) 발생.
*   **해결 과정:** 환자 ID와 날짜를 기준으로 복합 인덱스(Composite Index)를 생성하고 쿼리를 튜닝함.
*   **결과:** API 응답 속도를 기존 대비 약 XX% 단축하여 성능 최적화 달성.
