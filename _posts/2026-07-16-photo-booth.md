## 💡 프로젝트 개요
2026년 성년의 날을 기념하여 멋쟁이사자처럼 을지대-가천대 연합이 공동 운영한 오프라인 스냅 사진관 행사 지원 웹 서비스입니다. 촬영된 사진을 클라우드에 업로드하고, 참가자가 고유 QR 코드를 통해 본인의 사진을 스마트폰으로 다운로드할 수 있는 시스템을 구축했습니다.

## 📌 핵심 기여도
*   사전 방문자 등록, 관리자용 접수 현황 조회 등 행사 운영에 필요한 백엔드 라우팅 및 API 서버 구축
*   Java `MultipartFile` 인터페이스를 활용한 다중 이미지 파일 업로드 로직 구현
*   AWS S3 버킷 권한 설정을 통한 정적 이미지 스토리지 연동 및 파일 URL 반환 기능 개발

## 🔥 트러블슈팅 및 회고
**1. 고해상도 다중 이미지 업로드 시 메모리 초과(OOM) 방지**
*   **문제:** 부스 관리자가 고해상도 스냅 사진 원본 파일 4~5장을 한 번에 업로드할 때, Spring Boot 내장 톰캣의 업로드 용량 제한에 걸리거나 서버 메모리가 급증하는 문제 발생.
*   **해결:** 
    1. `application.yml` 파일에서 `spring.servlet.multipart.max-file-size` 및 `max-request-size` 속성을 행사 환경에 맞게 50MB 상향 조정.
    2. 파일을 서버 메모리나 로컬 디스크에 쌓아두지 않고, InputStream을 통해 AWS S3로 다이렉트 스트리밍 업로드(Direct Stream Upload) 되도록 로직을 수정함.
*   **결과:** 서버 부하를 최소화하여 200팀 이상의 사진 데이터를 안정적으로 처리하고 원활한 다운로드 환경을 제공함.
