## 💡 프로젝트 개요
대학생들이 챙겨야 할 교내활동, 행사, 자격증, 대회 등 파편화된 이력들을 한곳에서 기록하고, 추후 이력서 형태의 포트폴리오로 쉽게 추출할 수 있도록 돕는 경력 관리 웹 플랫폼입니다.

## 🏆 성과
*   **멋쟁이사자처럼 13기 교내 아이디어톤 2등 수상**

## 📌 핵심 기여도
*   경력 관리 플랫폼의 백엔드 데이터베이스 설계(ERD) 및 사용자별 활동 내역 CRUD API 구현
*   Swagger를 활용한 REST Docs 명세서 작성으로 프론트엔드 팀과의 협업 속도 향상
*   AWS EC2 기반의 서버 인프라 구축 및 배포 자동화(CI/CD) 초기 환경 세팅

## 🔥 트러블슈팅 및 회고
**1. 한정된 시간 내 프론트엔드와의 협업 효율성 극대화**
*   **문제:** 해커톤(아이디어톤)의 특성상 짧은 시간 안에 기획 변경과 기능 추가가 빈번하게 발생하여, 프론트엔드 팀원이 API 변경 사항을 실시간으로 파악하지 못해 병목 현상이 발생함.
*   **해결:** 스프링 부트에 `Springdoc-openapi(Swagger)`를 도입하여 코드 수정 시 API 명세서가 실시간으로 자동 업데이트되도록 구성함. 또한, 프론트엔드 팀이 서버 배포 전에도 개발을 진행할 수 있도록 Mock API 데이터를 사전에 제공함.
*   **결과:** API 소통 비용을 획기적으로 줄여 제한 시간 내에 목표했던 모든 핵심 기능을 완성하고 성공적으로 시연을 마칠 수 있었음.
